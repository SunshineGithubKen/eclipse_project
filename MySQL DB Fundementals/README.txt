コース概要：
このコースで学習したことは、HTTPとフォーム処理の仕組みを利用して、シンプルなWebアプリケーションの作成方法、オブジェクト指向プログラミング原則に触れ、クラスやインスタンスを扱うための文法を学ぶ。また、ファイルの読み書き、MySQLを題材に、データベースとは何か、複数のテーブルを結合してデータを検索や集計する方法を習得。


The Course Outline:
This course teaches how to create a web application using HTTP and forms. You'll learn object-oriented programming basics, including working with classes and instances. File handling is also covered. Lastly, the course covers MySQL, providing an understanding of databases and how to combine tables to search and analyze data.
