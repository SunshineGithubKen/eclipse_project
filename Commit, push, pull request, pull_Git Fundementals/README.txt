コース概要：

このコースでは、Gitという最も需要の高いバージョン管理ツールの使い方を学びます。
基本的なGitのコマンド操作から、Gitと連携するGithub（またはGitlab）の操作、
コミット履歴を分岐させるブランチ操作、コミット履歴を移動するチェックアウト、
分岐したコミット履歴を統合するマージ操作、そしてGithub上でマージを依頼するプルリクエストの方法まで学習します。

このコースの目的は、Gitを使ったバージョン管理の基本的なスキルを身につけることです。
実践的な演習問題を通して、GitHubを使ったマージ操作やプルリクエストによって、実際の開発現場での業務にも役立つスキル取得を目指しています。


Course Outline：
In this course, you will learn how to use Git, the most in-demand version control tool. You will start with the basic Git commands and move on to operating Github (or Gitlab) to collaborate with Git. You will learn how to branch commit history, move commit history using checkout, merge commit history branches, and request merge on GitHub using pull requests.

The goal of this course is to acquire the fundamental skills of version control using Git. Through practical exercises and projects, you will aim to obtain skills that are useful in actual development scenarios, such as merging operations and pull requests using GitHub.
